# MediView Scripts

Helper scripts for easy setup and management.

## 📜 Available Scripts

### Installation
- **install-backend.bat** - Install all backend dependencies
  ```bash
  scripts\install-backend.bat
  ```

### Server Management
- **start-backend.bat** - Start the backend server
  ```bash
  scripts\start-backend.bat
  ```

- **start-backend-simple.bat** - Start backend (alternative method)
  ```bash
  scripts\start-backend-simple.bat
  ```

## 🚀 Quick Start

1. **First Time Setup:**
   ```bash
   scripts\install-backend.bat
   ```

2. **Start Server:**
   ```bash
   scripts\start-backend.bat
   ```

3. **Stop Server:**
   Press `Ctrl + C` in the terminal

## 📝 Notes

- All scripts are Windows batch files (.bat)
- Run from project root directory
- Backend will run on port 5000
- Check docs/ for detailed guides
